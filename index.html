display: flex

Display: flex; é a primeira propriedade que precisa ser declarada dentro do container desejado para que o Flexbox funcione! Algumas propiedades como flex-direction, justify-content e align-content já vem com os valores padrão:

Propriedade: Valor:
Flex-direction row
justify-content flex-start
align-content flex-start

Exemplo: display: flex;

Este exemplo deve ser entendido como um começo, agora com esse elemento declarado somos capazes de adicionar várias outras prioridades para a organização dos itens (agora flexsíveis).
