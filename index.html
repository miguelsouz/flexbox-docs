Flex-direction:

Essa propriedade serve para dizer em qual direção o container deseja impilhar os itens flex. Recheada de alguns valores.



Exemplo: <Flex-direction: column>

Valor que deixa empilha os itens flexíveis verticalmente (de cima para baixo):



Exemplo: <Flex-direction: column-reverse>

Valor que empilha os itens flexíveis verticalmente (mas de baixo para cima):


Exemplo: <Flex-direction: row>

valor que empilha os itens flexíveis horizontalmente (da esquerda para a direita):



Exemplo: <Flex-direction: row-reverse>

Valor que empilha os itens flexíveis horizoltalmente (agora da direita para a esquerda):



Flex-wrap

A flex-wrap propriedade especifica se os itens flexíveis devem ser agrupados ou não.

Os exemplos abaixo possuem 11 itens flexíveis, para melhor demonstrar a flex-wrappropriedade:

Exemplo: <Flex-wrap: wrap>

O wrap: valor que especifica que os itens flexíveis serão agrupados se necessário:



Exemplo: <Flex-wrap: nowrap>

O nowrap: valor especifica que os itens flexíveis não serão agrupados (este é o padrão):



Exemplo: <Flex-wrap: wrap-reverse>

O wrap-reverse: valor que especifica que os itens flexíveis serão agrupados se necessário, na ordem inversa;
